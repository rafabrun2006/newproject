Como Funciona?

1 - Você cadastra um usuario e senha
2 - Acessa o sistema com este usuario e senha criado
3 - Cadastra Receitas, Despesas, Fornecedores
3 - Usuario pode alterar o perfil, clicando no menu perfil na parte superior do sistema
4 - Usuario pode vizualizar seu saldo total que fica no canto superior da tela.

Como Instalar o sistema?

Requisitos

1 - Banco de Dados Mysql
2 - Servidor Xampp ou Wamp

Instalação

1 - Copie a pasta financeiro para dentro do seu diretorio do localhost
2 - Importe o script de banco ( SQL ) que se encontra neste pacote atraves do phpmyadmin

Utilização

1 - Acesse o sistema com o seguinte endereço http://localhost/financeiro (caso tenha colocado na raiz do servidor)
